## Overview

The Document Vault is a secure document storage and sharing system that allows you to:
- Store important documents organized by category
- Share documents with third parties (bankers, auditors, etc.) securely
- Track who accessed your documents and when
- Manage document versions

---

## Getting Started

### Accessing Document Vault

1. Log in to ZenithBooks
2. Navigate to **Document Vault** from the sidebar menu
3. Or click on the **Document Vault** widget on your dashboard

---

## Uploading Documents

### Step 1: Upload a Document
1. Click **"Upload Document"** button
2. Select a file from your device
   - **Supported formats:** PDF, Images (JPG, PNG), Office Documents (Word, Excel)
   - **Maximum file size:** 50 MB per file
   - **Total storage:** 5 GB per user
3. Select a **Category** from the dropdown
4. (Optional) Add a description
5. Click **"Upload"**

### Categories Available
- Income Tax (ITR, Form 16, etc.)
- GST (Returns, Certificates, etc.)
- MCA (Incorporation, MOA, AOA, etc.)
- Registrations & Licenses (PAN, Aadhaar, UDYAM, etc.)
- Policies & Insurance
- Personal Documents (RC, Driving License, Passport, etc.)
- Banking & Financial
- Legal Documents
- Property & Real Estate
- Compliance & Certifications
- Contracts & Agreements
- Financial Statements & Reports
- Payroll & HR
- Others

### File Requirements
- ‚úÖ PDF files
- ‚úÖ Images: JPG, PNG
- ‚úÖ Office Documents: Word (.doc, .docx), Excel (.xls, .xlsx)

## Sharing Documents

### Creating Share Codes

**Share codes allow third parties (like bankers, auditors) to access specific document categories.**

#### Step 1: Create Share Code
1. Navigate to **"Share Codes"** from Document Vault menu
2. Click **"Create Share Code"**
3. Enter a name (e.g., "Housing Loan - HDFC Bank")
4. Enter or generate a secret code (minimum 8 characters)
5. Select categories to share (checkboxes)
6. (Optional) Add a description
7. Click **"Create"**

#### Important Notes:
- ‚ö†Ô∏è **Copy the share code immediately** - it won't be shown again!
- Share codes automatically expire after **5 days**
- You can create multiple share codes for different purposes
- Each code can have different categories shared

#### Example Use Cases:
- **Housing Loan:** Share Income Tax, GST, MCA, Banking categories
- **Business Loan:** Share GST, MCA, Banking, Financial Statements
- **Tax Audit:** Share Income Tax, GST, MCA, Registrations

### Managing Share Codes
- View all your share codes on the Share Codes page
- See expiry countdown for each code
- View access count (how many times accessed)
- Click **"Access Logs"** to see detailed activity
- Delete codes to immediately revoke access

### Sharing with Third Parties
1. After creating a share code, copy it
2. Share the following with the third party:
   - Share code (e.g., "HOUSE2024")
   - Access URL: `https://zenithbooks.in/vault/access`
   - Expiry information (5 days from creation)

## Third-Party Access (For Recipients)

### Accessing Shared Documents

1. Visit `https://zenithbooks.in/vault/access`
2. Enter the share code provided by the document owner
3. Click **"Access Documents"**
4. View and download documents from shared categories
5. **Note:** You can only see categories that were shared with you

### What Third Parties Can See
- ‚úÖ Documents in shared categories (latest versions only)
- ‚úÖ Download and view documents
- ‚úÖ Expiry countdown
- ‚ùå Documents in non-shared categories
- ‚ùå Document version history
- ‚ùå Other user's documents

## Access Logs

### Viewing Access Logs
1. Navigate to **"Access Logs"** from Document Vault menu
2. View all document access and downloads
3. Use filters to:

## Storage Management

### Viewing Storage Usage
- Storage usage is shown on the main Vault page
- See total used vs. total limit (5 GB)
- View storage breakdown by category
- Warnings appear at 80%, 90%, and 95% usage

### Storage Warnings
- **80% full:** Warning notification
- **90% full:** Critical warning notification
- **95% full:** Urgent warning notification

### Freeing Up Space
- Delete unused or old documents
- Delete old document versions
- Delete documents from categories you no longer need

## Troubleshooting

### "File size exceeds limit"
- Maximum file size is 50 MB per file
- Try compressing the file or splitting it

### "Storage limit exceeded"
- Your total storage limit is 5 GB
- Delete unused documents or old versions
- Check storage breakdown by category

### "Share code expired"
- Share codes expire after 5 days
- Create a new share code if needed
- Contact the document owner for a new code

### "Invalid share code"
- Check for typos (codes are case-sensitive)
- Verify the code hasn't expired
- Contact the document owner to verify the code

### Can't find a document
- Try using the search function
- Check different category tabs
- Verify the document wasn't deleted
