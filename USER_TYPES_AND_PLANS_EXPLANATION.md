**User Type** and **Subscription Plan** are TWO DIFFERENT things in ZenithBooks:

1. **User Type** (`userType`) - Selected during signup
   - Indicates WHO you are (your role/identity)
   - Options: `business` or `professional`
   - Cannot be changed after signup (without admin intervention)

2. **Subscription Plan** (`subscriptionPlan`) - Purchased separately
   - Determines WHAT features you can access
   - Options: `freemium`, `business`, or `professional`
   - Can be upgraded/downgraded anytime

## ğŸ”‘ Important Points

1. **User Type Restrictions**
   - **Business users** can subscribe to: Freemium, Business, or Professional Plan
   - **Professional users** can ONLY subscribe to: Professional Plan
   - Professionals need Professional Plan to manage multiple client businesses
   - Business Plan does not include Client Management (only Professional Plan has this)

2. **Professional Profile**
   - Only available to users with `userType: "professional"`
   - Independent of subscription plan
   - Allows professionals to be discovered in "Find a Professional" directory

3. **Client Management**
   - Only available with `professional` subscription plan
   - Only available to users with `userType: "professional"`
   - Allows managing multiple client business accounts from one dashboard
   - Essential feature for CAs and tax consultants managing multiple clients
   - This is why professionals MUST have Professional Plan

4. **Upgrading/Downgrading**
   - **Business users** can upgrade: Freemium â†’ Business â†’ Professional
   - **Business users** can downgrade plans (with feature restrictions)
   - **Professional users** must stay on Professional Plan (cannot downgrade to Business/Freemium)
   - User type cannot be changed after signup (admin only)

5. **Feature Access**
   - Feature access is determined by subscription plan
   - Menu items are shown/hidden based on `subscriptionPlan`
   - Upgrade alerts are shown on locked features
